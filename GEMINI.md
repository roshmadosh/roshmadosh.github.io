# Style Guide

- Font-family: Space Grotesk, and as a fallback Segoe UI
- Color scheme/theme: Assume I always want the color scheme in dark mode. The page background color should never be darker than #36454f. Adjust the background color to match that of the color scheme.
- Dotted line-breaks between posts in blogs.html
- Every page should have @docs/raining-cats-and-dogs.jpg added as an image just below the site title and navigation menu.
